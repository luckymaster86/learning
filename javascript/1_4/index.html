Методы работы с массивами

Метод POP

Вырезает последний элемент массива

<script>
myColors[1] = "green";
myColors.pop();
document.write("<br>"+myColors);

Метод PUSH

Добавляет последний элемент массива

<script>
myColors.push('blue');
myColors.push('yellow');
document.write("<br>"+myColors);
</script>

Методы shift и unshift

Добавляет и соответственно удаляет 1 элемент массива

<script>
document.write("<br>"+myColors);
myColors.shift();
document.write("<br>"+myColors);
myColors.unshift("pink");
document.write("<br>"+myColors);
</script>

Метод splice

Вырезаем часть массива. 1 параметр - индекс элемента с которого вырезаем. 2 параметр - сколько вырезаем.

Всегда возвращает массив.

<script>
myColors.splice(1,1);
document.write("<br>"+myColors);
</script>

Метод join

Выводит элементы массива в строку. Если указан парамет - он будет разделителем

<script>
var s = myColors.join('`````');
document.write("<br>"+s);
</script>

Метод split

Позволяет строку разрезать на массив строк

<script>
var str = "Саша,Маша,Даша,Лена";
var users = str.split(",");
for(var i = 0; i < users.length;i++){
	document.write("<br>"+users[i]);
}
</script>

Метод reverse

Переупорядочевает массив в обратном порядке

<script>
document.write(users.reverse());
</script>

Метод sort

Сортирует элементы массива

Если нужно отсортировать не как строки - нужно написать функцию сортировки.

<script>
document.write(users.sort()+"<br>");
var newArray = new Array(34,234,23,6,1,14,75);
document.write(newArray.sort());
function compareNumbers(n1,n2){
	if(n1==n2) return 0;
	if(n1>n2){
		return 1;
	}
	else{
		return -1;
	}
}
document.write("<br>"+newArray.sort(compareNumbers));
</script>
